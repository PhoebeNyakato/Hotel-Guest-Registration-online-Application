# Hotel-Guest-Registration-online-Application

//# Documentation for Hotel Guest Registration online Application ## 

Introduction This project is a guest registration platform for a hotel. 

## Installation 
1. Clone the repository 

2. Open `index.html` in your browser 
## Usage - Fill in your details in the form - Click **Submit** to register 

## Features
 - Collects guest information
 - Supports multiple room types 
 - Responsive design ## Storage 
 - Data is stored in a MySQL database

 ## File Structure
 - `index.html`: Main landing page
 - `prices.html`: Displays room rates 
 - `Guest_reservation.php`: Handles guest reservation form 
- `thank_you.html`: Confirmation page after registration 

## Technologies Used
 - HTML5 
 - CSS3 
 - PHP
